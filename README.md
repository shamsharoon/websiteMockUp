### Installation
1. **Clone the repository:**
    ```sh
    git clone https://github.com/shamsharoon/trash-tech.git
    ```
2. **Navigate to the project directory:**
    ```sh
    cd trash-tech
    ```
3. **Install dependencies:**
    ```sh
    npm install
    # or
    yarn install
    ```
4. **Start the development server:**
    ```sh
    npm run dev
    # or
    yarn run dev
    ```
